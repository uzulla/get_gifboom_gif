getgifboomgif
==============

えらい取得が面倒くさい、GifboomのGifのURLを、GifboomのポストのURLから取得します。

# NOTFOUND?

Gifboomは一時期Gif生成をとりやめていたことがあります。
その時期にポストされたデータはGifがありません。
（URLは取得できますが、DLできません）

# command

```
$ getgifboomgif http://gifboom.com/x/ae600e65
or
$ getgifboomgif ae600e65
```
